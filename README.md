# requisicoes-post-express
🚀 Projeto de Requisições POST com Express!
Este projeto foi desenvolvido como parte de um trabalho acadêmico sobre ambientes de desenvolvimento de software. O objetivo principal é demonstrar o funcionamento do método HTTP POST para envio de dados entre cliente e servidor.

🛠️ Tecnologias UtilizadasNode.js: 
Ambiente de execução JavaScript.
Express.js: Framework para criação de rotas e servidores web.
HTML/CSS: Interface para o usuário interagir com o servidor.

📝 O que foi implementadoConfiguração do Servidor: 
Servidor rodando na porta 3000.
Rota POST: Endpoint /api/post-example configurado para receber dados em formato JSON e de formulários.
Persistência Visual: Interface com estilo Glassmorphism para facilitar o teste da rota.
Middleware: Uso de express.json() e urlencoded para tratar as informações recebidas no req.body.

🚀 Como rodar o projeto?
Clone o repositório.
No terminal, execute npm install para instalar as dependências.
Inicie o servidor com node app.js.
Abra o index.html no navegador e teste o envio do formulário.
